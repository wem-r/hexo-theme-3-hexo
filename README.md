## Installation

```bash
git clone https://github.com/wem-r/hexo-theme-3-hexo.git themes/3-hexo
```

edit `_config.yml` and change :
```yaml
theme: 3-hexo
```

## Update
```bash
cd themes/3-hexo
git pull
```
